# UAV Logistics Simulator (Текстовий симулятор дрона)

Проєкт автономного логістичного симулятора, написаний на Python з використанням об'єктно-орієнтованого програмування (ООП). Розроблено повністю на мобільному пристрої в межах програми самоосвіти.

## 🚀 Функціонал програми
* **Клас Drone**: Моделює поведінку БПЛА (координати, заряд батареї, вантажопідйомність, логіка переміщення за Мангеттенською метрикою).
* **Клас Package**: Керує параметрами вантажу (вага, точки збору та доставки, динамічна зміна статусів).
* **Game Loop**: Інтерактивне консольне меню для керування симуляцією в реальному часі з захистом від некоректного введення даних (`try/except`).

## 🛠 Технологічний стек
* **Мова**: Python 3.x (Intermediate level)
* **Концепції**: ООП (Інкапсуляція, взаємодія об'єктів), Валідація даних, Обробка винятків.
* **Стандарт стилю**: PEP 8 (код перевірено автоматизованими лінтерами).

## 📊 Як запустити проєкту
Для запуску не потрібно встановлювати важкі програми. Проєкт працює в будь-якій хмарі (наприклад, Google Colab або Replit):
1. Скопіюйте код з файлу `main.py`.
2. Вставте в онлайн-інтерпретатор Python та натисніть **Run**.
3. Керуйте дроном за допомогою цифрових команд у консолі.
